# spanish-flashcards

A Spanish language learning app built with Vue 3. Presents flashcards for verbs and nouns across multiple tenses, randomizing which side of the card appears first.

## How it works

Each flashcard has four steps:
1. Word (Spanish or English, randomized)
2. Word + translation
3. Example phrase (one side)
4. Example phrase (other side)

Press **Next** to advance through steps or load a new card. Press **Previous** to return to the last card.

## Data

Flashcard data lives in `data/source.ts` and is generated by `scripts/generate-source.ts`. The generator produces conjugations for all verbs across five tenses (present, preterite, imperfect, future, conditional) and writes example phrases for each.

To regenerate the data after editing the script:

```sh
npx tsx scripts/generate-source.ts
```
